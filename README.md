## Self Host

Clone Repo

```bash
git clone ReCaree/flypixel
```

install Dependency

```bash
npm install
```

Run It

```bash
npm run dev
```

Go To [127.0.0.1:3000/public/index.html](127.0.0.1:3000/public/index.html)
